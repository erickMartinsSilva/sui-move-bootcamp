Exercício que consiste numa lista de afazeres (to-do list) com funções CRUD.

Conceitos praticados:
- Declaração e manipulação de structs via referências mutáveis;
- Definição de objetos com seus próprios UIDs via key e store;
- Criação de instâncias com base em contextos de transação;
- Transferência de ownership via função transfer.
